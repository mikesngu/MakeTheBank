## User Stories ##

**Log incomes and Expenses**
* As a __customer__ , I need to be able to add multiple expenses to see my monthly/yearly exepnditure *
* As a __customer__, I need to be able to add both expenses and income so i can see a net expenditure for a given time period*
* As a __customer__, I need to be able to write a description on each transaction so i can understand it at a later date*
* As a __customer__, I need to be able to manually adjust my transaction amount to account for errors*

## Categorize transactions ##
* As a __customer__, I need to be able to place my transactions into categories to track which category has the highest expenditure*
* As a __customer__, I need to be able to edit my transactions category incase i believe it fits another category better*
* As a __customer__, I need to be able to create new categories which may be more niche*

## Visualize Spending over time ##
* As a __customer__, I need to be able to view the date of each transaction when i click on it*
* As a __customer__, I need to be able to view my spending at different intervals such as a month or in a year. This should be both a numerical number and a graph*
* As a __customer__, I should be able to view my spending in each category at different time intervals*
